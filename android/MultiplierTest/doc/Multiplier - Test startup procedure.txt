---------------------------------------
* Multiplier - Test startup procedure *
---------------------------------------

What should be installed before you start:
	- Eclipse
	- Android support (updated Android SDK Manager)
	- EcFeedForAndroid plugin


- Create virtual Android device: 
	Android Virtual Device Manager > button Create... > fill parameters > button OK
		Example of parameters: 
			Target Android 4.4.2 - API Level 19
			CPU/ABI: ARM(armeabi-v7A)
			Hardware keyboard present: checked
			Skin: WVGA800
			RAM: 512
			VM Heap: 64
			Internal storage: 200

- Switch automatic compilation on:
	 Window > Preferences > Android > Build > Unselect checkbox: Skip packaging and dexing until export launch. > OK

- Import Multiplier project:
	File > Import > Android > Existing Android Code into Workspace > Next > Root directory (select your_repository/ecfeed.samples/android/Multiplier) > Ok > Finish

- Import MultiplierTest project: 
	File > Import > Android > Existing Android Code into Workspace > Next > Root directory (select your_repository/ecfeed.samples/android/MultiplierTest) > Ok > Finish

- Implement necessary Android code:
	In Package Explorer select: MultiplierTest > src > multiplier.ect (double click)
	In multiplier.ect window: Structure > select node: com.testify.multiplier.test.TestClass > Click Implement node icon (top right) > 
	Traffic light icon of TestClass node should go green. > There should be no compilation errors.

- Start your Virtual android virtual device: 
	Click icon Android Virtual Device Manager > select your Virtual device > click Start... button > click Launch button > 
	close Android Virtual Manager > wait till device starts > unlock device

- Run tests: 
	In multiplier.ect > select testMethod(..) > Click Test online button > click OK > click OK > You should see a Progress information Dialog with running test.
